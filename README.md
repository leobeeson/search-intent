# README

## Installation

1. Git clone...
2. Create virtual env:

    ```bash/powershell/cmd
    python3 -m venv .venv
    ```

3. Activate virtual env:

    ```bash/zsh
    source .venv/bin/activate
    ```

    ```powershell
    .venv\Scripts\Activate.ps1
    ```

    ```cmd
    .venv\Scripts\activate.bat
    ```

4. To contribute to the package, run from the project's root directory:

    ```bash/zsh
    pip install -e .
    ```

5. ...
